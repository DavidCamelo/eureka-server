## [Eureka Server](https://eureka.davidcamelo.com)
[![Docker Image CI / Kubernetes Deployment CD](https://github.com/DavidCamelo/eureka-server/actions/workflows/deploy.yml/badge.svg)](https://github.com/DavidCamelo/eureka-server/actions/workflows/deploy.yml)
### Overview
This project is based on Spring Boot and serves as a service registry for microservices within the application ecosystem. It utilizes Netflix Eureka to enable service discovery, allowing microservices to register themselves and discover other services dynamically.
